    PROCESS
    ======

     %utl_submit_py64("
     from shapely import wkt;
     poly = wkt.loads('POLYGON ((1 1, 2 1, 2 2, 1 2, 1 1))');
     pt = wkt.loads('POINT(0 0)');
     dist=poly.distance(pt);
     print dist;
     ");
